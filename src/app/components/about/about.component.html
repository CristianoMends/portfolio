Quem sou?

Desde criança, sempre tive interesse em tecnologia. mas, foi no SENAC, em 2022, que pude ver isso como uma carreira. No curso de Programador de Sistemas, tive acesso a uma base sólida de conhecimento em lógica de programação, além de conceitos de Java e SQL. Atualmente sou estudante de Engenharia de Software na Universidade Federal do Ceará, e estou sempre buscando aprimorar e adquirir mais conhecimentos na área.

Tenho experiência em criação de APIs Rest usando Java 17 com Spring Boot 3, incluindo testes unitários com JUnit e Mockito, documentação com OpenApi e Swagger UI, autenticação e autorização utilizando JWT e Spring Security para segurança de endpoints. Utilizo Spring Data JPA para realizar CRUD em bancos de dados relacionais como PostgreSQL, e Flyway Migrations para manter o versionamento do banco.

Além de desenvolvimento Backend, tenho experiência em desenvolvimento frontend usando Angular e React, focando na criação de interfaces responsivas e interativas. Além disso, possuo habilidades em consumir APIs para integrar dados e funcionalidades nos aplicativos que desenvolvo.

Estou aberto a novas oportunidades onde posso aplicar minhas habilidades e continuar aprendendo e crescendo profissionalmente.

Se tiver uma boa oportunidade que corresponda às minhas competências e experiência, fique á vontade para me contatar. Estou pronto para contribuir positivamente, seja em um ambiente corporativo ou em projetos individuais.
